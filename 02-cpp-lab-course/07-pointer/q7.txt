1. Enter the number of array first, then input each element of the array, and find the only duplicated element in the array (Output using Pointer).
    Sample 1:
    5
    1 3 5 1 2 
    Ans:
    1

    Sample 2:
    4
    4 5 1 5
    Ans:
    5 

2. Enter the number of array first, then input each element of the array, and output the maximum and minimum values of the sum of any two elements in the array (Output using Pointer).
    Sample 1:
    7 
    1 2 3 4 5 6 7
    Ans:
    Max: 13
    Min: 3

    Sample 2:
    3
    0 100 1000
    Ans:
    Max: 1100
    Min: 100

3. Enter the number of sets of winning numbers.
    Then enter the corresponding eight-digit winning number.
    Then enter the number that want to redeem.
    If there is a winning prize (at least three numbers), output the matching number, otherwise output "NO".
    Sample:
    3 (How many winning numbers)
    12345678 (First winning number)
    00001234 (Second winning number)
    87654321 (Third winning number)
    87655678 (The number wanting to redeem)
    Ans:
    5678
