# Java Programming Undergraduate Coursework

This folder showcases the fundamental skills learned throughout the Java Programming course.

While the original exercise codes are unavailable, the final project has been included to demonstrate practical application of the course concepts. 

The final project is a fully implemented **Tetris game** developed in Java. 
The implementation heavily references the tutorial available at: https://www.youtube.com/watch?v=N1ktYfszqnM&t=5319s

This project highlights the use of Java programming fundamentals, including object-oriented design, event handling, and graphical user interface development.
