# Python Programming Undergraduate Coursework

This folder documents the learning progression in Python Programming during the undergraduate course.

- Subfolders 01–04: Each contains a .txt file with the problem statements corresponding to the implemented Python programs.
- Subfolders 05–06: Contain the midterm and final projects, with topics chosen freely by the students.
- Note: The Snake game project references the implementation from https://github.com/clear-code-projects/Snake.

This folder serves as a comprehensive record of exercises, projects, and practical applications of Python programming concepts.
